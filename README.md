# ACHARYA - Adaptive AI Teaching Engine

ACHARYA is an AI-powered teaching system that dynamically adapts its teaching style based on student understanding.

## 🚀 Features

- AI-generated lessons
- Student answer analysis
- Automatic understanding detection
- Dynamic teaching style switching
- Socratic questioning mode
- Adaptive slide explanation mode
- Built using Google Gemini API

## 🛠 Tech Stack

- Python
- Streamlit
- Google Gemini API
- JSON-based reasoning engine
