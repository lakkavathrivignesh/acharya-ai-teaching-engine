# ACHARYA – Adaptive AI Teaching Engine

ACHARYA is an AI-powered adaptive learning system built using Streamlit and Google Gemini API.

## 🚀 Features

- AI-generated lesson
- Student understanding analysis
- Adaptive explanation system
- Slide-based learning
- Modern UI with gradient styling

## 🧠 How It Works

1. AI explains Newton's First Law.
2. Student answers conceptual question.
3. AI analyzes understanding level.
4. Adaptive slides are generated based on student level.

## 🛠 Tech Stack

- Python
- Streamlit
- Google Gemini API
